Types of HTML tags....

There are mainly two types of HTML tags:
  • Container tag
  • Empty tag
  1. Container tags: The HTML tags which need to be closed are called Container tags.
  2. Empty tags: The HTML tags which need not to be closed are called Empty tags.

    Introduction to tables in HTML...

    Tables are mainly used to display data into a beautiful and in arranged way
    to help the reader to read and extract data fill in the document easily.
    You already know some basic concepts about table that are used in the Microsoft Excel.
    Now, we are going to make table in webpage using HTML tags and we create our webpage more beautiful
    using HTML with help of tables.

    Firstly, tables are made up of rows and columns.
    Secondly, in tables horizontal lines make rows while, verticle lines makes columns.
    Finally, the place where horizontal and verticle lines intersect to form rectangle shape. This rectangle shape
    is known as Cell. As you also call them smallest unit of the table.
    1. TABLE: TABLE tag in HTML code is used to make the table. It is most important tag while you make tables.
    2. TH(table heading):TH tag in HTML used to insert data as a heading in the rows to display.
    3. TR(table row tag):TR tag is used in HTML to make rows in tables that we display on our webpage.
    4. TD(table data tag):TD tag is used in HTML to insert data in the rows to display.
